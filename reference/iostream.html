iostream¶

edrixs.iostream.write_emat(emat, fname, tol=1e-12)[source]¶

Write the nonzeros of a matrix to file. The first line is the number of nonzeros, and the following lines are the nonzero elements. This file will be read by ed.x, xas.x or rixs.x.

Parameters
emat2d complex array

The array to be written.

fnamestr

File name.

tolfloat

Precision.

edrixs.iostream.write_tensor(tensor, fname)[source]¶

Write \(n\) -dimension numpy array to file, currently, \(n\) can be 1, 2, 3, 4.

Parameters
tensor\(n\) d float or complex array

The array needs to be written.

fnamestr

edrixs.iostream.write_tensor_nonzeros(tensor, fname, tol=1e-06)[source]¶

Write nonzeros of the elements of a \(n\) -dimension numpy array to file, currently, \(n\) can be 1, 2, 3, 4.

Parameters
tensor\(n\) d float or complex array

The array needs to be written.

fnamestr

File name.

tolfloat

The precision, the elements whose absolute value is large than tol will be written.
